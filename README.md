# 🚀 Booking Pro System

**Booking Pro** — bu zamonaviy restoranlar yoki xizmat ko'rsatish markazlari uchun mo'ljallangan, rezervatsiyalarni boshqarish tizimi. Loyiha foydalanuvchilarga qulay interfeys va administrativ filtrlash imkoniyatlarini taqdim etadi.



## ✨ Xususiyatlari
- 📱 **To'liq Responsive:** Har qanday qurilma (mobil, planshet, desktop) uchun moslashtirilgan.
- 💾 **Local Storage:** Ma'lumotlar brauzer xotirasida saqlanadi, sahifa yangilanganda o'chib ketmaydi.
- 🔍 **Aqlli Filtrlar:** Rezervatsiyalarni "Bugun", "Kelajakda" va "Hammasi" kategoriyalari bo'yicha saralash.
- 🚫 **Double-booking Protection:** Bir xil vaqtga ikki marta band qilishni oldini olish tizimi.
- 🎨 **Modern UI:** Glassmorphism va minimalizm uslubidagi dizayn.

## 🛠 Texnologiyalar
- **HTML5:** Semantik struktura (`header`, `main`, `footer`).
- **CSS3:** Modern Grid, Flexbox va Fixed positioning.
- **JavaScript (Vanilla):** DOM manipulyatsiyasi va mantiqiy jarayonlar.
